# Task 2 – Exploratory Data Analysis & Business Intelligence

## Internship
ApexPlanet Software Pvt. Ltd. – Data Analytics Internship

## Objective
To uncover patterns, trends, and relationships in the dataset using exploratory data analysis and business-focused insights.

## Dataset
Cleaned sales transaction dataset prepared during Task 1.

## Analysis Performed
- Descriptive statistics for sales, profit, quantity, and discount
- Business questions analysis using pivot tables
- Sales analysis by customer segment, state, and month
- Multivariate analysis to study discount vs profit relationship
- Calculation of Average Order Value (AOV)
- Static dashboard mock-up highlighting key KPIs

## Tools Used
- Google Sheets

## Key Insights
- Consumer segment contributes the highest sales
- Sales vary significantly across months and regions
- Higher discounts tend to reduce profitability
- Average Order Value helps understand customer spending behavior
